# StuntLytics Brand Logo Implementation

The production logo is based on the approved generated concept and has been rebuilt as a responsive transparent vector for product use.

## Concept
- Medical cross: healthcare and public-health trust
- Rising trajectory: child growth monitoring and program improvement
- Analytics bars: regional intelligence and measurable follow-up
- Blue: clinical confidence and data clarity
- Mint/teal: health, prevention, and human-centered care

## Applied assets
- `components/BrandLogo.tsx` — responsive product wordmark component
- `public/stuntlytics-logo.svg` — full horizontal wordmark
- `public/stuntlytics-mark.svg` — standalone icon
- `public/icon.svg` — browser icon source
- `public/apple-icon.png` — Apple touch icon
- `public/icon-light-32x32.png` — light UI favicon raster
- `public/icon-dark-32x32.png` — dark UI favicon raster
- `public/brand/stuntlytics-logo-concept.png` — original generated visual concept reference

The logo is applied automatically wherever `BrandLogo` is used, including the editorial landing navigation/footer and workspace sidebar.
