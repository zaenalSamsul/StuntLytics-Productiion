# StuntLytics Design System

## Direction

**Modern Minimalism + Blue Mint Clinical Premium**

Clinical enough to feel trustworthy, modern enough to feel premium, and human enough to feel comfortable.

## Core tokens

| Token | Value | Usage |
|---|---:|---|
| Primary | `#2563EB` | Primary actions, active navigation, key series |
| Health Mint | `#14B8A6` | Positive health actions, secondary series, AI accent |
| Background | `#F7FAFC` | Application canvas |
| Surface | `#FFFFFF` | Cards, menus, inputs |
| Surface Blue | `#F5F9FF` | Supporting information |
| Surface Mint | `#F1FCF9` | Positive / calm health context |
| Text Primary | `#0F172A` | Headings and critical values |
| Text Secondary | `#475569` | Body copy |
| Text Muted | `#64748B` | Supporting metadata |
| Border | `#E2E8F0` | Standard boundaries |
| Success | `#16A34A` | Stable / positive status |
| Warning | `#D97706` | Needs attention |
| Danger | `#DC2626` | Critical / destructive only |

## Typography

- Heading stack: Plus Jakarta Sans → Inter → system sans
- Body stack: Inter → system sans
- Heading weights: 600–700
- Body weights: 400–500

## Radius

- Small: 8 px
- Medium: 12 px
- Large: 16 px
- XL: 20 px
- 2XL: 24 px

## Interaction

- Fast: 120–160 ms
- Normal: 180–240 ms
- Slow: 280–360 ms
- Reduced-motion preferences are respected globally.

## Accessibility

- Status uses icon + label, never color alone.
- Main shell interactive targets are at least 44 px.
- Focus-visible ring is globally defined.
- Mobile sidebar has explicit open/close controls and overlay label.
- AI insight includes a non-diagnostic disclaimer.
